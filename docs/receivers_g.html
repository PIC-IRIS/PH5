Receivers_g¶

The receivers group holds trace data, metadata related to trace data, and links to the miniPH5_#####.ph5 files containing timeseries data.

Das_g_xxxxx¶

Holds data from a given data logger or station. “xxxxxx” is based off serial number of data aquisition system (DAS).

Data_a_xxxx¶

The data array holds binary time series or textural data, usually in 32 bit floats or 32 bit integers. This data originated from DAS raw files.

Das_t¶

The das table holds information about the data from a given data logger. This table links Data_a_xxxx to which raw file it came from and in the case of a multiple channel DAS which channel a given.
